# Code for Leow, et al., Multimodal spatial omics reveals pro-invasive N-glycosylation in early human placentation

## Directories:
- R: contains code for analyses of DBDP glycans, EVT glycans and transcripts, and decidual proteomics and glycoproteomics datasets
- Python: contains code for aligning N-glycan images with histology scans, and for obtaining mean intensities per region from N-glycan images
- Matlab: contains code for glycan-enzyme co-occurrence analysis
